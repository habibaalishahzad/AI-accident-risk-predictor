# AI Accident Risk Predictor

A simple C++ program to predict accident risk based on driving conditions.

## Features
- Takes speed, weather, traffic, and road condition as input
- Calculates risk score 
- Returns: High Risk, Medium Risk, or Low Risk

## Logic Used
- Foggy weather: +40 risk
- Rainy weather: +30 risk  
- Heavy traffic: +20 risk
- Slippery road: +30 risk
- Speed > 80: +50 risk
